# Files app

Отвечает за файловое хранилище:

- загрузка файлов
- удаление
- переименование
- публичные ссылки

## Структура

### api.py

API для работы с файлами

### serializers.py

Валидация:

- загрузка файла
- изменение данных

### services.py

Логика:

- сохранение файлов
- генерация ссылок
- проверка доступа
