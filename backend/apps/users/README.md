# Users app

Отвечает за управление пользователями:

- регистрация
- аутентификация
- роли (администратор / пользователь)

## Структура

### api.py

Точки входа API (DRF views):

- регистрация
- логин
- список пользователей
- удаление пользователя

### serializers.py

Валидация входных данных и преобразование в JSON

### services.py

Бизнес-логика:

- создание пользователя
- удаление пользователя
- назначение ролей
