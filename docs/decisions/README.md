# Архитектурные решения

Здесь хранятся короткие записи о решениях, которые влияют на кодовую базу.

Рекомендуемый шаблон:

- Контекст
- Решение
- Почему так
- Последствия

Именование файлов:

- `YYYY-MM-DD-short-title.md`

Примеры будущих решений:

- структура монорепозитория;
- выбор модели пользователя;
- организация файлового хранилища;
- стратегия деплоя.
