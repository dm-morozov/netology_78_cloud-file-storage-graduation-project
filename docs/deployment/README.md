# Deployment

В этом разделе хранится все, что связано с запуском проекта и окружениями.

## Что хранить в этой папке

- инструкции по локальному запуску
- список переменных окружения
- docker-конфигурацию и правила запуска контейнеров
- описание dev- и prod-окружений
- CI/CD-пайплайны
- инструкции по публикации и обновлению

## Возможные будущие файлы

- `local-development.md`
- `environment-variables.md`
- `docker.md`
- `ci-cd.md`
- `production.md`

## Правило для раздела

Если документ отвечает на вопрос, как проект запускать, собирать, поставлять
или сопровождать, ему место в этой папке.
