# Docs Hub

Эта папка организована так, чтобы её было удобно открывать в Obsidian и быстро понимать состояние проекта.

## Навигация

- [[project/overview|Обзор проекта]]
- [[project/status|Текущий статус]]
- [[project/roadmap|Roadmap]]
- [[architecture/README|Архитектура]]
- [[api/README|API]]
- [[tasks/todo|Backlog]]
- [[tasks/in-progress|Задачи в работе]]
- [[tasks/done|Выполненные задачи]]
- [[decisions/README|Архитектурные решения]]
- [[netology/README|Материалы Netology]]
- [[workspace/README|Рабочая зона]]

## Как использовать

- Новые идеи и требования сначала фиксировать в `tasks/todo.md`.
- Активную работу переносить в `tasks/in-progress.md`.
- После завершения задачи переносить её в `tasks/done.md`.
- Решения по структуре, моделям, API и деплою фиксировать отдельными заметками в `decisions/`.
