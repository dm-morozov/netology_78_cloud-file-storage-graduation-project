# Netology

В этом разделе хранятся формальные требования дипломного проекта и материалы, которые приходят из курса. Здесь лучше не смешивать требования с вашими внутренними техническими решениями.

## Что хранить в этой папке

- исходное описание задания
- требования к backend-части
- требования к frontend-части
- заметки по консультациям и уточнениям от преподавателя
- ссылки на внешние материалы курса

## Текущие файлы

- [task-description.md](task-description.md) - Полное описание дипломного задания
- [backend-requirements.md](backend-requirements.md) - требования к backend
- [frontend-requirements.md](frontend-requirements.md) - требования к frontend

## Правило для раздела

Если документ описывает, "что требует курс", он должен лежать здесь.
Если документ описывает, "как именно мы это реализуем", его лучше переносить
в `architecture/`, `api/` или `deployment/`.
