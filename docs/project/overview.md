# Обзор проекта

## Идея

`My Cloud` — веб-приложение с пользовательским файловым хранилищем, административной панелью и SPA-интерфейсом.

## Цели первой версии

- Реализовать аутентификацию и регистрацию пользователей.
- Дать пользователю доступ к собственному файловому хранилищу.
- Дать администратору доступ к управлению пользователями и их файлами.
- Подготовить проект к развёртыванию на `reg.ru`.

## Предполагаемый стек

- Backend: `Python`, `Django`, `PostgreSQL`
- Frontend: `React`, `Redux`, `React Router`
- Infra: `Nginx`/Django static serving, файловое хранилище на сервере, деплой на `reg.ru`

## Принципы структуры

- Код и документация разделены.
- Решения фиксируются отдельно от задач.
- Статус проекта должен быть понятен без чтения всего репозитория.
