# Workspace

Эта папка предназначена для временных рабочих файлов:

- черновиков по архитектуре;
- промежуточных заметок по задаче;
- файлов, которые помогают быстро восстановить контекст проекта;
- временных артефактов, которые не нужно коммитить.

## Правила

- Можно создавать любые подпапки и markdown-файлы.
- Содержимое папки, кроме этого `README.md`, игнорируется через `.gitignore`.
- Если временная заметка становится важной, её нужно перенести в `project/`, `tasks/` или `decisions/`.
